- What: Supports the `Edit -> Spell` feature

## How to Install GNU Aspell
### Windows
Install using `scoop`:
```
scoop bucket add main
scoop install main/aspell
```

### macOS
Install using `brew install aspell`.

### Linux
Install using `sudo apt install aspell` or other methods.
