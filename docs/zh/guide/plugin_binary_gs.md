## 如何安装Ghostscript
### Windows
前往[Ghostscript官网](https://www.ghostscript.com)下载安装包并按照默认路径安装到系统C盘。

如果用户没有按照默认路径安装，则需要手动将`gs.exe`所在目录添加到PATH中。

### macOS
使用`brew install ghostscript`安装。

### Linux
使用`sudo apt install ghostscript`等方式安装。
