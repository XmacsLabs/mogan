# 202_53: `工具->统计...` 显示方式修改
## 如何测试
菜单栏`工具->统计...`，检查弹窗是否正确显示字符数、单词数、行数。
如果不确定是否正确显示，可复制全文至WPS Word，单击左下角`字数: ...`，检查Mogan和WPS的统计是否相同。

在Mogan中运行以下命令执行测试：
```
xmake run 202_53
```
