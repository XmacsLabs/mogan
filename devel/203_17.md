# [203_17] LaTeX 导入支持lbrack/rbrack

### 如何测试
以LaTeX形式导入 `\( \mathrm{d}z = \frac{1}{a}\left\lbrack  {\frac{1}{{e}^{y + \mathrm{{az}}} + 1}\mathrm{\;d}x + \frac{-{e}^{y + \mathrm{{az}}}}{{e}^{y + \mathrm{{az}}} + 1}\mathrm{\;d}y}\right\rbrack \)` 应该能正常渲染

## 2026/1/12

### What
doc2x识别图片输出的latex中包含\left\lbrack \right\rbrack等latex语法，而mogan中未支持这些，支持的是\left[ \right]等语法，因此出现渲染问题

本次修复让渲染器的映射处理函数支持\lbrack和\rbrack

### How
在 latex 语法映射处理函数 `latex_command_to_tree (tree t)` 中增加对"lbrack""rbrack"的识别即可处理：
```cpp
tree
latex_command_to_tree (tree t) {
	// ...
  if (is_large_delimiter (t, dtype)) {
    // ...
    if (s == "lbrack") s= "[";
    if (s == "rbrack") s= "]";
    // ...
  }
	// ...
}
```
