当前设置搜索替换的快捷键如下

| 功能                        | Linux/Windows      | macOS                             |
| --------------------------- | ------------------ | --------------------------------- |
| **创建选择区域**            | `Ctrl+？`          | `Command+？`                      |
| **插入通配符 x**            | `Ctrl+1`           | `Command+1`                       |
| **插入通配符 y**            | `Ctrl+2`           | `Command+2`                       |
| **插入通配符 z**            | `Ctrl+3`           | `Command+3`                       |
| **上一个匹配项**            | `↑` 或 `Page Up`   | `↑` 或 `Page Up`                  |
| **下一个匹配项**            | `↓` 或 `Page Down` | `↓` 或 `Page Down`                |
| **第一个匹配项**            | `Home`             | `Home` (或 `Fn+←`)                |
| **最后一个匹配项**          | `End`              | `End` (或 `Fn+→`)                 |
| **下一个匹配项 (F3)**       | `F3`               | `F3` (可能需 `Fn+F3`)             |
| **上一个匹配项 (Shift+F3)** | `Shift+F3`         | `Shift+F3` (可能需 `Fn+Shift+F3`) |
| **向前循环匹配 (搜索)**     | `Ctrl+Enter`       | `Command+Return`                  |
| **向后循环匹配 (搜索)**     | `Enter`            | `Return`                          |
| **替换当前匹配项**          | `Enter`            | `Return`                          |
| **替换所有匹配项**          | `Ctrl+Enter`       | `Command+Return`                  |
