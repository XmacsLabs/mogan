# 202_84 修复了页面尺寸更改时相应的界面显示错误

## 如何测试
1. 点击页面尺寸按钮，观察弹出菜单的最后一个按钮是否显示为自定义尺寸…
2. 点击自定义尺寸…，更改宽度与高度为5cm x 5cm，点击确认，观察页面尺寸按钮上显示的文字被更改了为5cm x 5cm
3. 点击文档->页面，观察页面种类的选项是否为5cm x 5cm
4. 点击页面种类的下拉窗，观察是否有5cm x 5cm选项，并在选择a4之后再点开下拉窗，这时5cm x 5cm已经不存在了
5. 重复第二步到第四步，更改高度与宽度，设置为10cm x 10cm，观察是否准确.
