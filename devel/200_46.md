# [200_46] Mogan STEM macOS goldfish自动打包脚本

## 如何测试
```bash
goldfish packages/macos/package.scm
```
需要在macOS系统中安装goldfish，方法如下
1. 克隆goldfish仓库
```bash 
cd ~/git
git clone git@github.com:MoganLab/goldfish.git
```
2. 编译goldfish仓库
```bash
xmake b goldfish
```
3. 将goldfish安装至/opt/goldfish
```bash
sudo xmake i -o /opt/goldfish --root
```
4. 配置goldfish全局可用
```bash
sudo ln -s /opt/goldfish/bin/goldfish /usr/local/bin 
```
