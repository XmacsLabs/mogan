# 202_90 给结构化跳转到下一个类似结构增加悬浮提示

## 如何测试
1. 打开TeXmacs/doc/about/mogan/stem.zh.tmu，点击 示例文档：我能用墨干做什么
2. 将鼠标停留在焦点工具栏的左侧，跳到上一处、跳到下一处标记的按钮，能显示快捷键
3. 点击在焦点工具栏上的标记的介绍（圆圈里面问号那个按钮），在 Structured navigation 板块中能观察到Ctrl+PageUp和Ctrl+PageDown快捷键
4. 打开TeXmacs/doc/manual/manu.editing.zh.tmu，在 结构化光标移动 模块中能看到2个Ctrl+PageUp和Ctrl+PageDown快捷键提示
5. 打开TeXmacs/doc/manual/manu.editing.en.tmu，在 Structured cursor movement 模块中能看到2个Ctrl+PageUp和Ctrl+PageDown快捷键提示
