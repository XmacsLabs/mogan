# 202_98 改进批注工具
## 如何测试
### 测试项1
1. 点击`插入->注记->折叠批注`，可以正常插入
2. 点击`插入->注记->展开批注`，可以正常插入
3. `Alt+:`插入展开批注
4. `Alt+;`插入折叠批注，检查是否是辅助窗口
5. 按照 [222_6.md](222_6.md) 的方法做测试，可以发现辅助窗口在多标签页中能够自动切换成对应的功能：
   + 搜索
   + 替换
   + 批注编辑器
