# 61_5 Gnuplot插件

## 2025/11/28 QT环境变量冲突
### What
由于本机和软件的QT环境变量冲突，导致 GUNPLOT 插件无法使用

## 如何测试
1. 按照gnuplot官方的文档安装gnuplot
   + Windows: 采用默认路径
   + macOS: `brew install gnuplot`
   + Linux: `sudo apt install gnuplot`
2. 打开`TeXmacs/tests/tmu/61_5_gnuplot.tmu`并按照文档测试
