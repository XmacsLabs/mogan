# [222_28] 禁用非Linux平台的内置截图工具

## 如何测试
1. 在 **Linux** 平台启动 Mogan
   - 按下 `Ctrl + Alt + X`
   - 内置截图工具应正常启动

2. 在 **Windows / macOS** 平台启动 Mogan
   - 按下 `Ctrl + Alt + X`
   - 不应触发任何内置截图行为
   - 不应出现报错或异常日志

3. 基本编辑、窗口、快捷键等其他功能不受影响

---

## 2025/01/16 
### What
在非 Linux 平台（Windows / macOS）上禁用内置截图工具及其全局快捷键，仅在 Linux 平台启用该功能。

### Why
- Windows 和 macOS 均已提供成熟、稳定的系统级截图工具  
- 内置截图工具在非 Linux 平台上属于冗余功能，且可能与系统快捷键或全局热键产生冲突  

### How
- 在 Qt GUI 初始化阶段增加平台判断
- 仅在 Linux 平台创建 `ScreenshotTool` 并注册全局快捷键
- 保持 Linux 平台行为不变，其他平台直接跳过相关初始化逻辑
