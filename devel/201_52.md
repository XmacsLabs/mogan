# [201_52] 新增 xmake 配置参数 debug_with_timestamp,实现封装的 debug 日志加上时间戳

## 如何测试
1. 默认-启用时间戳
```
xmake config --debug_with_timestamp=true
xmake b && xmake r stem
```

2. 不启用时间戳
```
xmake config --debug_with_timestamp=false
xmake b && xmake r stem
```

3. 查看启动日志，日志会带上时间戳,如下：
```
2026-01-12-11.15.57.736 TeXmacs debug-std, Use MuPDF render(1.25.1)
2026-01-12-11.15.57.750 TeXmacs debug-fonts, Try tfm ecrm10.tfm
2026-01-12-11.15.57.750 TeXmacs With linked TrueType support
```

## 2026/01/12 新增 xmake 配置参数 debug_without_timestamp,实现封装的 debug 日志加上时间戳

#### what
1. 新增 xmake 配置参数 debug_with_timestamp
2. 修改 tm_debug.cpp 添加时间戳格式化函数 get_formatted_timestamp()
3. 修改调试输出逻辑，在日志行首添加时间戳
4. 时间戳格式：yyyy-MM-dd-HH.mm.ss.SSS（如 2026-01-12-11.15.57.736）
5. 默认不启用时间戳，防止性能损耗，需显式配置启用
