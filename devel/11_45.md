# [11_45] 字体设为Noto CJK SC后，光标在部分字字符后倾斜

## 2025/11/18
### What

导致这个问题的原因初步判断是由于有些 unicode 字体有斜体熟悉，然后

```
// 分别对应: 以 真
SmartFont: 字符='<#4EE5>', 原斜率=0.1, math_kind=0, 字体类型=unicode:NotoSerifCJK-Regular.210@600
SmartFont: 字符='<#771F>', 原斜率=0.0933333, math_kind=0, 字体类型=unicode:NotoSerifCJK-Regular.210@600
```

### How
这里做的解决方案就是将哪些倾斜量比较小的 `< 0.15` 直接移正光标，
#### 测试
测试的话，就在 Noto CJK SC 字体下输入 "以|真|" 这两个字，|代表光标，看看是否倾斜就行
