## 2025/09/18

### What

在 ubuntu 发行版中,`XDG_SESSION_DESKTOP`环境变量 会显示 ubuntu 而不是 kde 或者 gnome

```bash
echo $XDG_SESSION_DESKTOP
#output
ubuntu
```

添加对环境变量的识别即可

```scheme
((== (xdg-dekstop-session) "ubuntu") "gnome")
```
